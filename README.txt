# Balan'z Klantenrapport

## Hoe gebruiken
1. Maak een GitHub repository aan (bijv. `balanz-klantenrapport`)
2. Upload deze drie bestanden:
   - balanz_klantenrapport.py
   - requirements.txt
   - README.txt
3. Ga naar [https://share.streamlit.io](https://share.streamlit.io)
4. Klik 'New app' → kies jouw repo → selecteer `balanz_klantenrapport.py`
5. Klik 'Deploy'

De app opent en je kunt:
- Een financieel PDF-rapport uploaden
- Automatische analyse bekijken
- Persoonlijke commentaar toevoegen
- Een professioneel PDF-klantenrapport downloaden
